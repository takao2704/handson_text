# LTEモジュール

## このページで確認するもの

LTEモジュールの認識、SIM挿入、APN設定、IP疎通確認を行います。

## 対象者

LTEモジュールを使ってSORACOM Airへ接続する受講者。

## 所要時間

30分

## このページの前提

- 対応するSORACOM IoT SIMが準備済み
- モジュールのドライバ、制御ツール、アンテナが準備済み

## 使用するもの

- LTEモジュール
- SORACOM IoT SIM
- アンテナ
- 接続確認用PCまたはデバイス

## 手順

### 1. デバイス認識を確認する

OSからモジュールが認識されていることを確認します。

### 2. APNを設定する

ハンズオンで利用するSORACOMのAPN、認証方式、ユーザー名、パスワードを教材に明記します。

### 3. IP疎通を確認する

```bash
ip route
ping -c 3 8.8.8.8
```

:::warning
SIM ID、IMSI、ICCIDなどの個別識別情報をスクリーンショットに含めないでください。
:::
