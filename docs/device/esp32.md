# ESP32

## このページで確認するもの

ESP32を利用する場合の開発環境、ファームウェア書き込み、通信確認の観点を整理します。

## 対象者

ESP32からセンサーデータを送信する受講者。

## 所要時間

20分

## このページの前提

- Arduino IDE、PlatformIO、または利用する開発環境がセットアップ済み
- USBシリアル接続でログを確認できる

## 使用するもの

- ESP32開発ボード
- USBケーブル
- センサーまたはサンプルデータ生成コード

## 手順

### 1. ボード認識を確認する

開発環境でESP32のポートが認識されていることを確認します。

### 2. サンプルスケッチを書き込む

シリアルモニタに起動ログが出力されることを確認します。

### 3. 通信方式を決める

Wi-Fi、LTEモジュール、ゲートウェイ経由など、ハンズオンで利用する通信方式を明記します。

## よくあるエラー

- USBシリアルドライバが入っていない
- 書き込みモードに入っていない
- ボーレート設定が合っていない
