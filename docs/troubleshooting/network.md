# ネットワーク接続

## 症状

- デバイスから外部へ通信できない
- SORACOMユーザーコンソールでSIMがオンラインにならない
- DNS名前解決に失敗する

## 確認順序

1. デバイスがLTEモジュールを認識しているか確認する
2. SIMが有効で、利用可能な状態か確認する
3. APN、認証方式、ユーザー名、パスワードを確認する
4. IPアドレスとデフォルトルートを確認する
5. DNS名前解決を確認する

## コマンド例

```bash
ip addr
ip route
ping -c 3 8.8.8.8
nslookup soracom.io
```

## 解決しない場合

受講者のSIM IDやIMSIを公開チャンネルに貼らず、イベント運営者に個別に共有します。
