# 後片付けチェックリスト

ハンズオン後に残りやすいリソースを確認するためのチェックリストです。

## SORACOM

- Harvest Dataを使った場合は、対象グループの設定と保存データを確認する
- Beam、Funnelを使った場合は、対象グループの転送設定を確認する
- 認証情報ストアに登録した認証情報を確認する
- SIMのグループ割り当てを確認する

## AWS

- AWS IoT Coreのルール、証明書、ポリシーを確認する
- Lambda関数、CloudWatch Logs、IAMロールを確認する
- S3バケットとオブジェクトを確認する
- Timestreamのデータベース、テーブル、保持期間を確認する
