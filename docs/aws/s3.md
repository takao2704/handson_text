# S3

## このページで確認するもの

ハンズオンで受信したIoTデータをAmazon S3へ保存する場合のバケット、キー設計、後片付けを整理します。

## 手順

### 1. バケットを作成する

イベント用の命名規則を決め、リージョンと公開設定を確認します。

### 2. オブジェクトキーを決める

```text
telemetry/year=YYYY/month=MM/day=DD/<YOUR_DEVICE_ID>.json
```

### 3. 保存結果を確認する

AWSマネジメントコンソールまたはCLIでオブジェクトが作成されていることを確認します。

## 後片付け

作成したオブジェクトとバケットを削除します。
