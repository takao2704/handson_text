# ハンズオン集

このページは、IoTデバイス、SORACOM、AWSに関する小さなハンズオンのカタログです。

各ハンズオンは独立して実施できる前提で整理します。共通のコース順は設けず、必要なアカウント、機材、課金ポイント、後片付けは各ハンズオンの冒頭に閉じ込めます。

## 選び方

| やりたいこと | ハンズオン |
|---|---|
| デバイスからSORACOMまでの疎通を確認したい | [デバイスからSORACOMへ送信](./01-device-to-soracom.md) |
| SORACOMからAWSへデータを渡したい | [SORACOMからAWSへ連携](./02-soracom-to-aws.md) |
| AWSに保存したデータを確認、可視化したい | [データの可視化](./03-visualize-data.md) |

## 書き足すときのルール

- そのハンズオンだけで必要な前提条件を書く
- 使うサービス、作るリソース、課金ポイントを書く
- サンプル値と実値を分け、秘密情報を含めない
- 後片付けと確認方法を書く
- 他のハンズオンに依存する場合は、必須依存ではなく代替入力も書く

:::tip
既存のハンズオンを「続きもの」として読めるようにしても構いませんが、単体で開いた読者が迷わない構成を優先します。
:::
