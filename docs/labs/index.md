# 目的別カタログ

このページは、IoTデバイス、SORACOM、AWSに関するハンズオンを「やりたいこと」から探すための入口です。

各ハンズオンは独立して実施できる前提で整理します。共通のコース順は設けず、必要なアカウント、機材、課金ポイント、後片付けは各ページの冒頭に閉じ込めます。

## 選び方

| やりたいこと | 入口 |
|---|---|
| デバイスやサンプルからSORACOMへ送信したい | [データを送る](../catalog/send-data.md) |
| 届いたデータを確認、保存、可視化したい | [データを見る](../catalog/view-data.md) |
| SORACOMからAWSへデータを渡したい | [クラウドへ連携する](../catalog/cloud-integration.md) |
| Raspberry Pi、ESP32、LTEモジュールを準備したい | [デバイスを準備する](../catalog/prepare-device.md) |
| 通信、証明書、権限の問題を切り分けたい | [ネットワークを確認する](../catalog/check-network.md) |
| 作成したリソースと課金対象を整理したい | [後片付け・課金確認](../catalog/cleanup-cost.md) |

## 書き足すときのルール

- そのハンズオンだけで必要な前提条件を書く
- 使うサービス、作るリソース、課金ポイントを書く
- サンプル値と実値を分け、秘密情報を含めない
- 後片付けと確認方法を書く
- 他のハンズオンに依存する場合は、必須依存ではなく代替入力やサンプルデータも書く

:::tip
ページを追加するときは、サービス名ではなく「読者が何をしたいか」を起点に配置します。
:::
