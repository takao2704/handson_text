# デバイスを準備する

ハンズオンで使うデバイス、通信モジュール、SIM、開発環境を確認するためのページです。

## この目的で使うページ

| ページ | 使う場面 |
|---|---|
| [Raspberry Pi](../device/raspberry-pi.md) | Linuxベースのデバイスで通信確認したい |
| [ESP32](../device/esp32.md) | マイコンからセンサーデータを送信したい |
| [LTEモジュール](../device/lte-module.md) | LTEモジュール、SIM、APNを確認したい |
| [SORACOM Inventory](../soracom/inventory.md) | デバイス管理の観点も扱いたい |

## 始める前に確認すること

- 利用するデバイスと通信方式が決まっている
- 電源、ケーブル、アンテナなどの周辺機器が揃っている
- デバイスログを確認できる

## 関連する目的

- 通信状態を確認する場合は[ネットワークを確認する](./check-network.md)
- データを送る場合は[データを送る](./send-data.md)
