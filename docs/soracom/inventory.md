# SORACOM Inventory

## このページで確認するもの

SORACOM Inventoryを利用する場合のデバイス登録、認証、管理対象情報を整理します。

## 手順

### 1. デバイス管理方針を決める

イベントでInventoryを使う目的を明確にします。

### 2. デバイスを登録する

デバイスID、認証方式、グループを確認します。

### 3. 管理情報を確認する

メタデータ、状態、接続履歴を確認します。
