# SORACOM Beam

## このページで確認するもの

デバイスからのHTTP、MQTT、TCPなどの通信をBeamで受け、クラウド側のエンドポイントへ転送する設定を整理します。

## 手順

### 1. 利用するプロトコルを決める

ハンズオンで使うプロトコル、ポート、転送先を明記します。

### 2. Beam設定を作成する

SORACOMユーザーコンソールでグループにBeam設定を追加します。

### 3. デバイスから送信する

デバイス側はSORACOMのエンドポイントに送信し、Beamがクラウド側へ転送することを確認します。

## よくあるエラー

- 転送先URLのパスが違う
- TLS設定が合っていない
- デバイスが所属するグループにBeam設定がない
