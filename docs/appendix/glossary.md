# 用語集

| 用語 | 説明 |
|---|---|
| SORACOM Air | IoT向けのセルラー通信サービス |
| SORACOM Beam | デバイスからの通信をクラウドへ転送するサービス |
| SORACOM Funnel | クラウドサービスへデータを直接連携するサービス |
| SORACOM Harvest Data | デバイスデータを簡易に保存、可視化するサービス |
| AWS IoT Core | IoTデバイスとAWSサービスを接続するマネージドサービス |
| Lambda | サーバー管理なしでコードを実行するAWSサービス |
| Timestream | 時系列データ向けのAWSデータベースサービス |
