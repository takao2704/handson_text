# 前提条件

## 必要なアカウント

| 種別 | 用途 |
|---|---|
| SORACOMアカウント | SIM管理、Harvest Data、Beam、Funnelなどの設定 |
| AWSアカウント | IoT Core、Lambda、S3、Timestreamなどの作成 |
| GitHubアカウント | 教材の参照、Issue、Pull Request、GitHub Pages運用 |

## 必要な機材

- Raspberry Pi、ESP32、LTEモジュールなどのIoTデバイス
- SORACOM IoT SIMまたは利用する通信モジュールに対応した回線
- USBケーブル、電源、アンテナなどの周辺機器
- ブラウザを利用できるPC

## 必要なローカルツール

- Node.js 20以上
- pnpm 11系
- Git
- テキストエディタ

## 課金確認

ハンズオンではSORACOMとAWSの有料リソースを作成する場合があります。各手順では以下を明記してください。

- 利用するサービス
- 想定される課金ポイント
- 無料利用枠の対象可否
- 後片付け手順

:::info
画面表示やメニュー名は、サービスのアップデートにより変更される場合があります。イベント前に最新画面で手順を確認してください。
:::
